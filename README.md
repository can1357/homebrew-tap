# homebrew-tap

Homebrew tap for [omp](https://omp.sh) — the oh-my-pi coding agent.

## Install

```sh
brew install can1357/tap/omp
```

`Formula/omp.rb` is regenerated automatically on every release by
`scripts/ci-update-brew-formula.ts` in
[can1357/oh-my-pi](https://github.com/can1357/oh-my-pi); edit it there, not here.
